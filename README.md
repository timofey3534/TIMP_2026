## 📑 Техническое задание (Темы №25)

В рамках проекта реализуется программный комплекс, объединяющий криптографические методы, численные алгоритмы и стеганографию.

### Ключевые технологии и алгоритмы:
* **Симметричное шифрование:** `AES` (Advanced Encryption Standard).
* **Хеширование:** `SHA-384`.
* **Численный метод:** Метод хорд (для решения нелинейных уравнений).
* **Стеганография:** Внедрение текстового сообщения в изображение.

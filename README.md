# 📂 Содержимое ветки `docs`

Данная ветка является изолированным хранилищем всей проектной документации, графических материалов и проектных схем. Она не содержит программного кода и служит единым источником актуальной информации о проектировании системы.

---

## 🏗 Структура репозитория в ветке

| Путь / Файл | Описание | Формат |
| :--- | :--- | :--- |
| `/diagrams` | Папка с исходниками диаграмм (draw.io, Visio и т.д.) | `.drawio`, `.png` |
| `stub_404.jpg` | Временная заглушка для документаций | `image/jpg` |

---

## 🔄 Порядок обновления
1. Все изменения в документацию вносятся **напрямую** в ветку `docs`.
2. При обновлении диаграмм необходимо заменять старые файлы новыми с **сохранением прежних названий**, чтобы ссылки в Wiki и основном README не ломались.
3. Исходники (редактируемые файлы) рекомендуется хранить в подпапке, чтобы можно было внести правки в логику схем позже.
